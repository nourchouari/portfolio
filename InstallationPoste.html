Projet : Installation et configuration d’un serveur LAMP

Dans le cadre de cette SAÉ, j’ai réalisé l’installation et la configuration d’un serveur LAMP (Linux, Apache, MySQL, PHP) sur une machine virtuelle Ubuntu MATE hébergée sur clé USB via VirtualBox.
L’objectif était de mettre en place un environnement complet pour héberger un site web dynamique :

  • Installation et configuration des services Apache, PHP et MySQL, ainsi que leurs modules de liaison.
  • Création d’une base de données MySQL et d’un site de test affichant du contenu dynamique via PHP.
  • Rédaction et exécution d’une feuille de tests unitaires pour vérifier le bon fonctionnement de chaque composant.
Le projet s’est conclu par une démonstration de l’environnement opérationnel et un oral en anglais présentant les étapes techniques, les choix réalisés et les difficultés rencontrées.
