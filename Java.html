Projet : Trunks – The Namek Quest

En équipe de trois, nous avons développé un jeu 2D type Terraria dans l’univers de Dragon Ball, en Java 21 et JavaFX. Ce projet mêlait conception logicielle, programmation orientée objet et gestion agile.
Fonctionnalités principales :

  • Déplacements en 2D type plateforme (saut, course, chute).
  • Carte dynamique en tuiles, inventaire et système d’artisanat.
  • Combats contre des ennemis et boss, interactions avec PNJ, amélioration d’objets.
  • Pouvoirs spéciaux via une barre de Ki.
  • Architecture MVC propre et modulaire pour assurer évolutivité et maintenabilité.
Outils et méthodes : GitHub (workflow main/dev), Trello (gestion agile en 3 sprints). Projet réalisé sur 7 semaines dans le cadre des SAEs de programmation, structures de données, UX et gestion de projet.
